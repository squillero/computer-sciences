PYTHON

Computer Sciences is now in [Python](https://en.wikipedia.org/wiki/Python_(programming_language))!

## References

The official Python's documentation can be found at <[https://docs.python.org](https://docs.python.org)>.

:exclamation: Python is being actively developed and some, important details may be different from what is described in text books.

## Programming Environment (IDEs)

:exclamation: You need a recent version of Python (check out the [**official distributions**](https://www.python.org/downloads/))

* :star: [**Visual Studio Code**](https://code.visualstudio.com/): a free and powerful editor loaded with IDE-like features — this is the **official** option for the course.

* [**PyCharm**](https://www.jetbrains.com/pycharm/): professional IDE — a free [educational license](https://www.jetbrains.com/community/education/#students) is available for Politecnico's students.
